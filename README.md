# pdf2img.py

## 环境配置
### Centos
```
pip install wand
pip install PyPDF2
yum install ImageMagick -y
```
### MAC
```
sudo pip install wand
sudo pip install PyPDF2
brew install ImageMagick
brew install ghostscript
```


## 运行
```
python pdf2img.py ur_filename
```

OR

```
python pdf2img.py ur_path/ur_filename
```
